# Connect — BGSU Volleyball

Recruit quick-questions page. Athletes answer three questions and leave their
contact info; submissions flow into the staff Google Sheet ("Connect Responses").

- Live: https://bgsuvolley.github.io/connect
- Stack: single static `index.html` (no build step), GitHub Pages
- Submissions: silent POST to a Google Form → linked Sheet
- To change where answers go: edit the `FORM` config block in `index.html`

Maintained with Claude.
